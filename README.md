# 饮食分享平台管理系统 java122

## 项目概述

饮食分享平台是一个集用户互动与菜品管理于一体的网络平台。本项目基于SSM框架和SpringBoot技术进行开发，实现了用户端与管理员端的功能模块。

## 技术栈

- SSM框架（Spring、SpringMVC、MyBatis）
- SpringBoot
- MySQL
- IDEA或Eclipse等Java开发环境
- JDK

## 功能模块

### 用户端

- 登录/注册
- 查看菜谱详细信息
- 查看系统公告
- 查看或添加收藏和笔记
- 查看饮食咨询
- 参与趣味答题

### 管理员端

- 用户端所有功能
- 增删改查菜品信息与分类
- 管理公告、收藏、笔记及点赞信息
- 管理趣味答题信息

## 系统角色

- 普通用户
- 管理员

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL
- Java版本：JDK 1.8及以上

## 部署步骤

1. 环境准备：确保MySQL和JDK版本符合要求。
2. 数据库部署：运行数据库目录下的`.sql`文件，完成数据库初始化。
3. 项目导入：使用IDEA或Eclipse等工具，将项目导入到开发环境。
4. 配置修改：根据实际情况修改配置文件中的数据库连接信息等。
5. 项目运行：启动应用，进行测试。
